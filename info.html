Finite Automata Visualization

What is this Project About

This project is a web-based Finite Automata Visualization tool that allows you to create, view, and simulate finite automata in an interactive and visual way.

  • Two different creation modes: Code Mode and Button Mode
  • Create and build both DFAs and NFAs
  • Stepwise simulation of automata on an input word
  • Interactive editing of states and transitions

Understanding Finite Automata

Finite automata are abstract computational models used to design and analyze the behavior of digital systems. They consist of a finite number of states and transitions between those states based on input symbols.

Deterministic Finite Automata (DFA)

A DFA is an automaton where each state has exactly one transition for each input symbol. DFAs are predictable and efficient, making them ideal for applications like lexical analysis and pattern matching.

Nondeterministic Finite Automata (NFA)

An NFA can have multiple transitions for the same input symbol or transitions without input (Îµ-transitions). Although they offer design flexibility, NFAs are typically converted to DFAs for actual execution.

How to Use this Project

This project features two modes for creating automata:

Code Mode

In Code Mode you can manually enter the FSM definition. You may either write your own code or copy one of the provided sample codes (DFA or NFA). This mode is ideal for users who prefer to work directly with the automata's code.

Button Mode

Button Mode provides a graphical interface where you can:

  • Add State: Click the "Add State" button to insert a new state on the canvas.
  • Add Transition: Use the "Add Transition" button to create a connection between states.
  • Delete Selected: Remove any selected element (state or transition) from the automata.
  • Update Selected: Modify properties (such as labels or state types) of the selected element using the update field.

Project Features & Additional Information

Beyond basic automata creation and simulation, this tool offers:

  • Intuitive interface with a dark theme for reduced eye strain
  • Support for both DFA and NFA with seamless switching between them
  • Step-by-step simulation that visually highlights state transitions
  • Clipboard functionalities to quickly load sample automata code in Code Mode

The project is continuously evolving to include more advanced features and enhanced usability.
